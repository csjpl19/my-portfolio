# Images des projets

Ajoute ici les captures de tes projets.

Chaque carte projet accepte au maximum 3 images. Dans `index.html`, place-les dans le bloc:

```html
<div class="project-slides">
  <img src="assets/projects/mon-projet-1.jpg" alt="Capture du projet">
  <img src="assets/projects/mon-projet-2.jpg" alt="Deuxieme capture du projet">
  <img src="assets/projects/mon-projet-3.jpg" alt="Troisieme capture du projet">
</div>
```

Formats conseilles: `.jpg`, `.png` ou `.webp`.
Ratio conseille: 16:10 ou 16:9.
